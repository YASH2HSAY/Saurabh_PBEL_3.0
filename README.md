# 📰 Fake News Detection using Machine Learning

A machine learning project that classifies news articles as **REAL** or **FAKE**
based on their text content, using NLP preprocessing and a TF-IDF + linear
classifier pipeline. Built as part of an internship project.

## Project Overview

Fake news spreads fast and is often written in a distinctive style — sensational
headlines, emotional language, vague sourcing, urgency cues ("SHARE BEFORE IT'S
DELETED"). This project trains a text classifier to pick up on those patterns
and flag likely fake articles.

**Pipeline:**
1. **Text cleaning** — lowercasing, URL/HTML/punctuation removal, stopword
   removal, lemmatization (`src/preprocess.py`)
2. **Feature extraction** — TF-IDF vectorization with unigrams + bigrams
3. **Classification** — Passive-Aggressive / Logistic Regression classifier
4. **Evaluation** — accuracy, precision/recall/F1, confusion matrix
5. **Deployment** — CLI prediction tool + optional Streamlit web demo

## Dataset

`data/news_dataset.csv` contains 600 labeled rows (300 REAL, 300 FAKE) with
columns `title`, `text`, `label`. It's programmatically generated
(`src/generate_dataset.py`) so the whole project runs end-to-end without
needing external downloads, and so it's not built from copyrighted articles.

**For a stronger, more realistic model**, swap in the public Kaggle
[Fake and Real News Dataset](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset)
(~45,000 real articles). Download `True.csv` / `Fake.csv`, add a `label`
column (`REAL` / `FAKE`), combine them into `data/news_dataset.csv` with the
same three columns, and rerun `train.py`. Everything else works unchanged.

## Setup

```bash
# 1. Clone the repo
git clone <your-repo-url>
cd fake-news-detection

# 2. Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt
```

## Usage

### 1. (Optional) Regenerate the sample dataset
```bash
python src/generate_dataset.py
```

### 2. Train the model
```bash
python src/train.py
```
This prints accuracy, a classification report, and a confusion matrix, then
saves the trained model to `models/fake_news_model.pkl`.

Train with a different classifier:
```bash
python src/train.py --model logistic_regression
```

### 3. Predict on new text
```bash
python src/predict.py --text "BREAKING: Secret documents PROVE officials lied about everything!"
```
Or run it interactively:
```bash
python src/predict.py
```

### 4. Launch the web demo (optional)
```bash
streamlit run src/app.py
```

## Results

On the included sample dataset, the model achieves very high accuracy because
the synthetic data has clearly separable style patterns. When trained on the
full real-world Kaggle dataset instead, expect a more realistic **~93–97%
accuracy** — that's the number worth reporting if you swap in real data.

| Metric | Score (sample dataset) |
|---|---|
| Accuracy | ~100% (synthetic data) |
| Precision / Recall / F1 | See `train.py` output |
